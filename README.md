# ishop
## Интернет магазин разработанный на базе собственного фрэймворка ##
## Что сделано:
### 1. Главная страница с отображаемыми товарами из БД, выщитываемые дисконты на товар, цена товара конвертируется в трех валютах по курсу из БД
### 2. Поиск по категориям включая вложенные категории, фильтр товаров, связанные товары, поиск товара по названию, корзина, пагинация и т.д.
### 3. Вход, регистрация. Если авторизован как админ, вход на админпанель ishop\admin . 
### 4. Админпанель. Разные настроенные виджеты для легкого ориентирования по админке, просмотр,редактирование и добавление: категорий, товаров, пользователей, валют,фильтров для товаров.
### 5. На сайте реализованно кэширование категорий и фильтров на 1час, при необходимости легко можно одной кнопкой почистить кэш категорий или фильтров.
### 6. Некоторые товары имеют модификации, которые можно просмотреть и выбрать на странице данного товара. Добавление и удаление товара реализуется без перезагрузки страницы.
### 7. Товары при применении фильтров отображаются так, чтобы в отображаемом товаре имелись все выбранные категории и фильтры.
